# Layout Components in MDX

MDX allows you to create sophisticated layouts directly in your markdown documents. This example demonstrates various layout components and techniques you can use to structure your content.

## Basic Two-Column Layout

<TwoColumnLayout>
  <Column>
    ### First Column

    This is the content for the first column. You can put any markdown or components here.

    - List item 1
    - List item 2
    - List item 3
    
    <Alert type="info">
      Info alerts work great in columns!
    </Alert>
  </Column>
  
  <Column>
    ### Second Column
    
    Content for the second column goes here. Columns are responsive and will stack on smaller screens.
    
    ```jsx
    function HelloWorld() {
      return <div>Hello, world!</div>;
    }
    ```
    
    ![Placeholder image](https://source.unsplash.com/random/400x300/?nature)
  </Column>
</TwoColumnLayout>
