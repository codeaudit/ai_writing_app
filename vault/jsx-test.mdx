# Testing JSX Code Blocks

This is a test file with various code block formats to ensure our MDX renderer handles them correctly.

## JSX Code Block

```jsx
function HelloWorld() {
  return <div>Hello, world!</div>;
}

export default HelloWorld;
```

## JavaScript Code Block

```js
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

## TypeScript Code Block

```typescript
interface User {
  name: string;
  age: number;
}

function greetUser(user: User): string {
  return `Hello, ${user.name}! You are ${user.age} years old.`;
}
```
